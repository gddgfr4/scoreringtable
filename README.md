# WA Scoring Table Profile

World Athletics Scoring Tables of Athletics 2025 revised edition のPDFから、以下のトラック種目を抽出して使う静的サイトです。

- 100m
- 200m
- 400m
- 800m
- 1500m
- 3000m
- 5000m
- 10000m

## できること

- 男女のWAスコア表を切り替え
- 8種目の自己ベストを入力してWA得点を表示
- 距離軸または対数距離軸で得点の折れ線グラフを表示
- 各自己ベスト得点を、他種目なら何分何秒相当か8x8表で表示

公式表に空欄がある点数は、前後の公式値から補間して ≈ を付けています。

## 使い方

index.html をブラウザで開くだけで動きます。ビルド作業や外部ライブラリは不要です。
